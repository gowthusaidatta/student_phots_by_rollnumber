## 🎓 Roll Number Format (Quick Reference)

```
23 MH 1A 49 01
│  │  │  │  └─── Sequential (01-99, A0-Z9)
│  │  │  └────── Branch Code (05=CSE, 49=IoT, etc.)
│  │  └───────── Dept/Year identifier
│  └──────────── Campus (MH/P3/A9)
└─────────────── Year (23=2023)
```

**To download all IoT students from 2023**:
- Enter: `23MH1A49`
- Gets: 23MH1A4901 to 23MH1A49Z9
